### Shadcn Themes

Shadcn is a javascript library that has virtually any component that you are going to need for your project, completely customizable.

You download components to your project with a command, and you’re free to do whatever you want with it.

Now, there’s one tiny issue with the library.

It uses a very specific structure of style variables to set colors, border radius etc., but there are many themes they provide.

After I noticed a need in the community for more themes, I decided to build a Shadcn themes library as well as a generator.

So if you’re using Shadcn, go ahead and use it. Completely free.

[Get Shadcn Themes](https://www.shadcn.studio)

This product is the first one that actually went well for me.
